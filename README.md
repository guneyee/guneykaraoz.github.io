# Guney E. - Personal Portfolio

Welcome to my personal portfolio website! This site showcases my work as a software developer and technology enthusiast.

## About This Site

This website is built using the [Feeling Responsive](https://github.com/Phlow/feeling-responsive) Jekyll theme, customized for my personal portfolio.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Portfolio Showcase**: Display your projects and work
- **Blog Integration**: Share your thoughts and technical insights
- **CV Download**: Easy access to download my resume
- **Social Media Links**: Connect with me on GitHub and other platforms
